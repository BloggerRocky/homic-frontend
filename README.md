# Homic 家庭云存储服务 - 前端

## 项目简介

Homic 家庭云存储服务是一个现代化的家庭云存储平台前端应用，基于 Vue 3 + Element Plus 构建，提供用户友好的界面和丰富的功能体验。

## 版本信息

**当前版本：v1.0.2**

## 技术栈

- **框架：** Vue 3.2.47
- **UI组件库：** Element Plus 2.2.36
- **路由：** Vue Router 4.1.6
- **状态管理：** Pinia 2.0.32
- **HTTP客户端：** Axios 1.3.4
- **构建工具：** Vite 4.1.4
- **样式：** Sass 1.59.2
- **主题：** 支持深色/浅色主题切换

## 核心功能

### 📁 文件管理
- **文件上传：** 支持多文件上传、拖拽上传、大文件分片上传
- **文件预览：** 支持图片、视频、音频、PDF、Office文档、文本文件预览
- **文件操作：** 重命名、移动、删除、下载、分享
- **文件夹管理：** 创建文件夹、文件夹导航、面包屑导航
- **文件分类：** 全部、视频、音频、图片、文档、其他分类浏览
- **搜索功能：** 文件名模糊搜索
- **排序功能：** 按名称、创建时间、修改时间、文件类型排序

### 👥 好友系统
- **好友管理：** 添加好友、删除好友、好友列表
- **好友搜索：** 根据用户名或邮箱搜索好友
- **好友申请：** 发送好友申请、处理好友申请
- **文件分享：** 向好友分享文件，支持永久有效分享

### 👨‍👩‍👧‍👦 家庭系统
- **家庭管理：** 创建家庭、加入家庭、家庭信息管理
- **家庭空间：** 家庭共享文件空间
- **关怀账号：** 为家庭成员创建关怀账号，简化操作界面
- **家庭头像：** 家庭头像上传和管理

### 📤 分享功能
- **链接分享：** 生成分享链接，支持访问码保护
- **分享管理：** 查看分享记录、取消分享
- **外部分享：** 无需登录即可访问分享内容

### 🗑️ 回收站
- **文件恢复：** 10天内删除的文件可恢复
- **永久删除：** 彻底删除不需要的文件

### ⚙️ 系统设置（管理员）
- **用户管理：** 查看用户列表、管理用户状态
- **文件管理：** 查看所有用户文件、系统文件统计
- **系统设置：** 系统参数配置

### 🎨 用户体验
- **主题切换：** 支持深色/浅色主题切换
- **响应式设计：** 适配不同屏幕尺寸
- **上传进度：** 实时显示上传进度和状态
- **空间统计：** 显示存储空间使用情况
- **右键菜单：** 便捷的右键操作菜单

## 主要页面

1. **登录页面** (`/login`) - 用户登录
2. **首页** (`/main/:category`) - 文件管理主界面
3. **我的分享** (`/myshare`) - 分享记录管理
4. **回收站** (`/recycle`) - 删除文件管理
5. **好友管理**
   - 我的好友 (`/friend/myFriends`)
   - 搜寻好友 (`/friend/search`)
   - 好友申请 (`/friend/requests`)
6. **家庭管理**
   - 家庭管理 (`/family`)
   - 家庭空间 (`/family/space`)
   - 关怀账号 (`/family/careAccount`)
7. **系统设置**（管理员）
   - 用户文件 (`/settings/fileList`)
   - 用户管理 (`/settings/userList`)
   - 系统设置 (`/settings/sysSetting`)

## 开发环境要求

- Node.js 16+
- npm 或 yarn

## 安装运行

```bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 生产环境构建
npm run build

# 预览构建结果
npm run preview
```
